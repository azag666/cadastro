Guia de Configuração no ManyChat 

            
Siga este passo a passo para configurar corretamente seu ManyChat e integrar com o AZAG System.

            
            

            

                
                

                    
 1. Configure o Gatilho (Trigger) do seu Fluxo

                    
O gatilho é o que "inicia" seu fluxo no ManyChat. Para que o AZAG System funcione corretamente com o Telegram, o gatilho deve ser configurado para quando o usuário optar pelo Telegram. Siga a imagem abaixo para configurar exatamente:

                    
                    
Captura de Tela 2025-07-24 às 04:21:58.png

                    
    
                        
  • No ManyChat, crie um novo fluxo ou edite um existente.
    • 
                        
  • Clique em "Add Trigger" (Adicionar Gatilho).
    • 
                        
  • Escolha a opção "User Custom Field" (Campo Personalizado do Usuário).
    • 
                        
  • Selecione o campo do sistema: **`Telegram Opt-in`**
    • 
                        
  • Defina a condição como: **`is True`** (é Verdadeiro).
    • 
                    

                    
Isso fará com que seu fluxo seja ativado sempre que um usuário confirmar a opção de receber mensagens pelo Telegram.

                


                
                

                    
 2. Defina o Click ID com a Última Entrada de Texto

                    
O `click_id` é essencial para o rastreamento individual de cada venda. A primeira ação no seu fluxo, logo após o gatilho, deve ser a de capturar a "Última Entrada de Texto" do usuário e salvá-la em um campo personalizado chamado `click_id`. Siga a imagem abaixo:

                    
                    
Captura de Tela 2025-07-24 às 04:25:29.png

                    
    
                        
  • Adicione um bloco de "Action" (Ação) logo após o gatilho.
    • 
                        
  • Dentro da ação, clique em "Set Custom Field" (Definir Campo Personalizado).
    • 
                        
  • Selecione ou crie o campo personalizado: **`click_id`** (tipo Texto).
    • 
                        
  • Defina o valor como: **`{{last_text_input}}`** (Última Entrada de Texto).
    • 
                    

                    
Isso garante que cada interação do usuário seja rastreada com um identificador único.

                


                
                

                    
 3. Importe o Fluxo Completo AZAG System no ManyChat

                    
Para facilitar sua integração, criamos um fluxo único que contém todas as requisições e ações necessárias para o AZAG System. **Este fluxo é um exemplo de como as requisições e ações podem ser organizadas, e você pode adaptá-lo às suas necessidades após a importação.** Clique no botão abaixo para importar este fluxo diretamente para o seu ManyChat:

                    
                    
                         Importar Fluxo Completo AZAG System
                    

                    
**Visualização do Fluxo Completo:**

                    

                        
                    

                    
Após clicar em "Importar Fluxo Completo AZAG System", o ManyChat será aberto com o fluxo carregado. Este fluxo já contém blocos de "Requisição Externa" para: **Ajustar Valor PIX**, **Gerar Pagamento (PIX)** e **Confirmar Pagamento**.

                    
**Importante:** Após a importação, é fundamental clicar no botão Publish (Publicar) no canto superior direito do ManyChat para salvar e ativar o fluxo.

                


                
                

                    
 4. Verifique e Crie os Campos Personalizados (Se Necessário)

                    
Campos Personalizados são como "gavetas" no ManyChat onde guardamos informações importantes sobre seus clientes (como o valor do PIX, o status do pagamento, etc.). **Quando você importa o fluxo completo, o ManyChat geralmente cria essas "gavetas" automaticamente para você.** Mas é sempre bom dar uma olhada para ter certeza de que está tudo certo!

                    
Para verificar, siga estes passos simples no seu ManyChat:

                    
    
                        
  1. No menu lateral esquerdo do ManyChat, clique em Settings (Configurações).
    2. 
                        
  2. Em seguida, clique em Custom Fields (Campos Personalizados).
    3. 
                        
  3. Você deve ver uma lista de campos. Verifique se os seguintes campos de usuário existem e se o "Tipo" (Type) está correto:
    4. 
                    

                    
    
                        
  • **valor em reais**: Tipo **Number**
    • 
                        
  • **valor da venda**: Tipo **Number**
    • 
                        
  • **click_id**: Tipo **Text** (Este é o campo que você definiu no Passo 2)
    • 
                        
  • **$.id**: Tipo **Text**
    • 
                        
  • **$.qr_code**: Tipo **Text**
    • 
                        
  • **$.status**: Tipo **Text**
    • 
                    

                    
                    
Captura de Tela 2025-07-23 às 19.03:53.png

                    
**Se algum campo estiver faltando, você pode criá-lo manualmente:**

                    
    
                        
  1. Na mesma tela de + New User Field, clique no botão azul + New User Field.
    2. 
                        
  2. No campo 'Field Name' (Nome do Campo), digite o nome exato do campo que está faltando (ex: `valor em reais` ou `$.id`).
    3. 
                        
  3. Em 'Field Type' (Tipo do Campo), escolha o tipo correto (use **Number** para valores numéricos e **Text** para IDs e QR Codes).
    4. 
                        
  4. Clique em 'Create' (Criar).
    5. 
                    

                    
**Importante:** Preste muita atenção aos nomes dos campos, incluindo o `$.` para os campos `$.id`, `$.qr_code` e `$.status`. O ManyChat é sensível a isso, mas geralmente aceita o `$.` sem problemas.
