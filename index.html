Guia Visual: Onde Encontrar os Códigos 

        
Este tutorial irá guiá-lo passo a passo para encontrar as informações necessárias para o cadastro.

        
        

        

            
            

                
 Nome do Cliente/Empresa

                
Este é o nome da sua empresa ou do cliente que você está onboardando. É um campo de texto livre para sua identificação interna.

                
                
**Onde Encontrar:** Basta digitar o nome que você deseja usar para identificar o cliente no sistema.

            


            
            

                
 Username do Bot Telegram

                
O username do seu bot no Telegram é crucial para que a pressel possa redirecionar os usuários para o bot correto.

                
                
**Onde Encontrar:**

                
    
                    
  1. Abra o Telegram e procure por **@BotFather**.
    2. 
                    
  2. Inicie uma conversa e digite `/mybots`.
    3. 
                    
  3. Selecione o bot que você deseja usar.
    4. 
                    
  4. O BotFather mostrará as informações do seu bot, incluindo o **Username** (ex: `@meubot_bot`). Copie este valor.
    5. 
                

            


            
            

                
 Token da PushinPay (do seu cliente)

                
Este é o token de autenticação da API do seu cliente na PushinPay. Ele permite que nossa API gere PIX diretamente na conta do seu cliente.

                
                
**Onde Encontrar:**

                
    
                    
  1. Acesse o painel da PushinPay do seu cliente.
    2. 
                    
  2. Procure por seções como "Configurações", "API", "Chaves de Acesso" ou "Desenvolvedores".
    3. 
                    
  3. O token de API geralmente é uma string longa (ex: "sk_live_XXXXXXXXXXXXXXXXXXXX"). **Copie o token de API secreto.**
    4. 
                    
  4. **Importante:** Este token é sensível. Não o compartilhe publicamente.
    5. 
                

            


            
            

                
 ID do Pixel da Meta (Facebook Pixel ID)

                
O ID do Pixel da Meta é usado para rastrear eventos no navegador e para a API de Conversões, garantindo o monitoramento de suas campanhas.

                
                
**Onde Encontrar:**

                
    
                    
  1. Acesse o Meta Business Manager.
    2. 
                    
  2. Vá para **Gerenciador de Eventos**.
    3. 
                    
  3. Selecione a conta de anúncios e o Pixel que você deseja usar.
    4. 
                    
  4. O **ID do Pixel** é um número que geralmente aparece no canto superior esquerdo ou nas configurações do Pixel. Copie este número.
    5. 
                

            


            
            

                
 Token da API de Conversões da Meta

                
Este token é essencial para enviar eventos de servidor para o Facebook, melhorando a precisão do rastreamento de suas conversões.

                
                
**Onde Encontrar:**

                
    
                    
  1. No **Gerenciador de Eventos** da Meta, selecione seu Pixel.
    2. 
                    
  2. Vá para a aba **Configurações**.
    3. 
                    
  3. Role para baixo até a seção **API de Conversões**.
    4. 
                    
  4. Clique em **"Gerar Token de Acesso"** ou "Gerar token de acesso". Copie o token gerado.
    5. 
                    
  5. **Importante:** Este token é sensível e deve ser tratado como uma senha.
    6.
